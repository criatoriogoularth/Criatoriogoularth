📤 Importar Árvore Genealógica

Faça upload do PDF da árvore genealógica para preencher automaticamente os dados da ave

📋 Como funciona:

  • 1. Faça upload do PDF da árvore genealógica (gerado pelo sistema)
  • 2. O sistema vai extrair os dados automaticamente
  • 3. Você pode revisar os dados antes de importar
  • 4. Os dados vão preencher a ave e toda sua árvore genealógica
  • 5. Se a ave já existir, os dados faltantes serão completados
